Kalkulator Sederhana
Aplikasi kalkulator sederhana ini dibuat menggunakan HTML, CSS, dan JavaScript. Kalkulator ini mendukung operasi dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian.

Fitur
- Antarmuka pengguna yang sederhana dan intuitif.
- Mendukung operasi dasar matematika.
- Menampilkan hasil kalkulasi secara real-time.
- Tombol backspace untuk menghapus karakter terakhir yang dimasukkan.
- Tombol clear untuk menghapus semua input.
